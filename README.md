# MLOps Course Project

## Система мониторинга и предсказания задержек запросов в облачной инфраструктуре

### О проекте
Разработка и развертывание в Yandex Cloud MLOps-пайплайна для предсказания задержек запросов (`query_latency`) в распределенной облачной системе.

### Технологический стек
- **IaC:** Terraform
- **Облачная платформа:** Yandex Cloud
- **Оркестрация:** Apache Airflow
- **Трекинг экспериментов:** MLflow
- **Хранилище данных:** Yandex Object Storage (S3)
- **API сервис:** FastAPI
- **Контейнеризация:** Docker
- **Оркестрация контейнеров:** Kubernetes (YMDB)
- **CI/CD:** GitHub Actions
- **Мониторинг:** Prometheus + Grafana
- **Алертинг:** Alertmanager + Telegram

### Структура проекта

├── infra/ # Terraform конфигурации
├── k8s/ # Kubernetes манифесты
├── dags/ # Airflow DAGs
├── src/api/ # FastAPI сервис
├── scripts/ # Вспомогательные скрипты
├── notebooks/ # Jupyter notebooks (EDA)
└── docs/ # Документация
